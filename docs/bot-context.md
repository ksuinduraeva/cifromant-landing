# Контекст бота Цифромант

Telegram-бот с платной подпиской. Считает нумерологические числа по дате рождения и имени, отправляет персональный AI-прогноз каждое утро подписчикам.

**Бот:** @number_day_bot (прод), @number_day_test_bot (тест)
**Репозиторий:** github.com/ksuinduraeva/numberDayBot
**Хостинг бота:** Railway (две среды: dev и prod)
**БД:** Supabase PostgreSQL (два проекта: dev и prod)

---

## Стек бота

- **Node.js** + **Telegraf** v4 — логика бота
- **pg** (node-postgres) + **Supabase** — PostgreSQL БД
- **OpenAI API** (gpt-5.4-mini-2026-03-17) — генерация прогнозов
- **Express** — сервер для вебхука YooKassa
- **node-cron** — рассылка в 08:00 по местному времени пользователя
- **YooKassa** — приём платежей

---

## Подписка и монетизация

- **29 ₽/день** — пробный день
- **99 ₽/неделя** + 1 день в подарок
- **299 ₽/месяц** + 7 дней в подарок
- Оплата через YooKassa, вебхук верифицируется обратным запросом к API

## Функционал бота

```
Главное меню:
├── 🔢 Мои числа → число дня рождения, число пути, самовыражения, души
├── 👥 Рассчитать другого → имя + дата → числа + deep link для шаринга
├── 📅 Число дня
│       ├── Нет подписки: личное число дня + upsell
│       └── Есть подписка: AI прогноз на день (кеш или генерация)
├── 📆 Прогнозы (подписчики) → месяц / год
├── 💑 Совместимость (подписчики) → AI-анализ пары
├── ⭐ Подписка → статус / оформление
└── ⚙️ Мои данные → дата рождения, имя, ФИО, часовой пояс
```

## AI-прогнозы — правила промптов

Все прогнозы генерируются через OpenAI (gpt-5.4-mini-2026-03-17).

### Что передаётся в промпт (ежедневный прогноз)
- Имя, число жизненного пути, число дня рождения
- Личный день/месяц/год, дата, день недели, сезон
- Вводный абзац числа жизненного пути из статичных текстов
- Полный текст числа дня рождения из статичных текстов
- Последние 3 прогноза с тем же числом дня (для исключения повторов)

### Ограничения для всех прогнозов (обязательные)
- Писать ТОЛЬКО на русском языке — ни одного английского слова
- Никаких обещаний будущего («произойдёт», «случится», «будет»)
- Никакой мистики, пафоса и эзотерического жаргона
- Никаких медицинских или финансовых советов
- Обращение на «ты», тепло и конкретно
- Называть числа пользователя в тексте («твоя тройка пути», «личный день 5»)
- Прогноз должен быть написан так, чтобы его нельзя было отправить другому — он отражает именно это уникальное сочетание чисел
- В каждом разделе показывать взаимодействие энергии числа дня с числом пути и числом рождения

### Структура ежедневного прогноза (5 разделов)
1. 🔮 Общий фон дня
2. 🌿 Что поддержит
3. ⚡ Где быть внимательнее
4. 🕯 Практика дня
5. 💭 Вопрос дня

### Структура месячного прогноза (5 разделов)
1. 🌙 Тема месяца
2. 🌱 Что сейчас в росте
3. 🎯 На чём фокус
4. ⚠️ Что учитывать
5. 💫 Вопрос месяца

### Структура годового прогноза (5 разделов)
1. 🌟 Главная тема года
2. 🚀 Возможности периода
3. 🔄 Что трансформируется
4. 🎯 Задача года
5. 💭 Вопрос года

### Совместимость — особые правила
- Перед генерацией — отдельный вызов API для склонения имени партнёра по падежам
- Мастер-числа (11/22/33) называть «мастер-число 33 пути», не «тридцать три пути»
- Имя партнёра упоминается не более одного раза в предложении
- Никогда не ставить рядом местоимение «его» и имя в родительном падеже

## Нумерологические расчёты

- **Число дня рождения** — день рождения (сводится к одному числу)
- **Число жизненного пути** — сумма всех цифр полной даты рождения
- **Число самовыражения** — числовое значение всех букв ФИО при рождении
- **Число души** — числовое значение гласных букв ФИО при рождении
- **Личный день/месяц/год** — персональные циклы, рассчитываются от даты рождения
- Мастер-числа (11, 22, 33) не сводятся дальше

---

## Статичные тексты (src/texts/index.js)

Для каждого числа (1–9, 11, 22, 33) есть:
- `birthdayNumberTexts` — описание числа дня рождения (на «ты»)
- `lifePathTexts` — описание числа жизненного пути (на «ты», 4 раздела)
- `birthdayNumberTextsOther` — то же, но для третьего лица
- `lifePathTextsOther` — то же, но для третьего лица
- `expressionNumberTexts` — число самовыражения
- `soulNumberTexts` — число души
- `todayNumberTexts` — личное число дня

Тексты: на «ты», гендерно нейтральные, Markdown (`*жирный*`).
Стиль: тёплый, конкретный, без мистики — описывают характер и природный стиль.

## Реферальная система

- Пользователь делится ссылкой вида `t.me/number_day_bot?start=share_TOKEN`
- Получатель открывает → видит числа друга → кнопка «Узнать свои числа»
- При первой оплате друга реферальщик получает бонус:
  - Неделя (99₽) → +1 день
  - Месяц (299₽) → +7 дней
- Правило «первый выигрывает» — повторные share-ссылки не перебивают первого

---

## Tone of voice продукта

- Нумерология — не мистика, а закономерности
- Тексты на «ты», тепло, без снисхождения
- Конкретно — не «энергия числа», а «для тебя с тройкой пути это проявляется как...»
- Без обещаний («произойдёт», «ждёт тебя») — только про текущее состояние и возможности
- Акцент на уникальность: каждый прогноз — только для этого человека, с его числами
