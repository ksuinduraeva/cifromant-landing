# Цифромант — Лендинг

Сайт нумерологического Telegram-бота **Цифромант**. Основная задача — SEO-трафик через статьи про нумерологию и конвертация в подписчиков бота.

**Публичное название:** Цифромант
**Слоган:** Цифромант — когда цифры становятся подсказками
**Сайт:** cifromant.ru
**Бот в Telegram:** @number_day_bot

## Продукт: что, зачем и для кого

**Что мы делаем:**
Цифромант — нумерологический инструмент на базе AI. Рассчитывает персональные числа по дате рождения и имени, генерирует уникальный прогноз на день/месяц/год и анализирует совместимость пар. Каждый прогноз написан именно для этого человека — с его числами, его характером, его текущим периодом.

**Зачем мы это делаем:**
Люди хотят понимать себя и происходящее вокруг. Нумерология даёт простой и доступный язык для этого. Мы убираем мистику и пафос — и делаем из нумерологии практичный инструмент самопознания, который работает каждый день.

**Для кого:**
- Преимущественно женщины 25–60 лет
- Интересуются нумерологией, астрологией, психологией, саморазвитием
- Хотят понимать себя и отношения глубже
- Ценят персонализацию — «специально для меня», а не «общий гороскоп»
- Открыты к практикам и рефлексии, но не хотят тяжёлой эзотерики
- Готовы платить за качественный ежедневный инструмент (29–299 ₽/месяц)

**Ключевое отличие от конкурентов:**
Не шаблонный гороскоп для всех — а прогноз, который учитывает уникальное сочетание чисел конкретного человека. Пользователь чувствует, что текст написан именно про него.

**Роль лендинга:**
Привлечь человека через поиск (он ищет «число жизненного пути» или «совместимость по числам») → дать ценную статью → познакомить с Цифромантом → отправить в бота.

## Стек лендинга

- **Next.js 15** (App Router) — фреймворк, SSG/SSR для SEO
- **TypeScript** — типизация
- **Tailwind CSS** — стили
- **Sanity** — headless CMS для статей (редактор на `/studio`)
- **Vercel** — хостинг, автодеплой из ветки `main`

## Структура

```
app/
  page.tsx                  # главная страница
  articles/
    page.tsx                # список статей
    [slug]/page.tsx         # отдельная статья
  studio/[[...tool]]/       # Sanity Studio (редактор статей)
sanity/
  schemas/article.ts        # схема статьи
  client.ts                 # клиент Sanity + функции запросов
sanity.config.ts            # конфиг Sanity
docs/
  bot-context.md            # полный контекст бота-близнеца
  promotion-plan.md         # план раскрутки
```

## Рабочий процесс

1. Код пишем локально в `/Documents/projects/landing/`
2. Пушим в `main` → Vercel автоматически деплоит на cifromant.ru
3. Статьи добавляются через cifromant.ru/studio (без кода)

## Страницы

```
/                — главная: hero, что умеет бот, CTA → Telegram
/articles        — список SEO-статей про нумерологию
/articles/[slug] — отдельная статья + CTA в конце → бот
/studio          — Sanity редактор (только для администратора)
```

## Дизайн

- Тёмный фон: `#0f0a1e`
- Акцентный цвет: фиолетовый (`purple-600` / `purple-400`)
- Шрифт: системный sans-serif
- Tone of voice: тёплый, конкретный, без мистики и пафоса — тексты на «ты»

## Правила контента

- Статьи пишутся на русском языке
- Каждая статья заканчивается CTA-блоком со ссылкой на бота
- Никакой мистики и эзотерического жаргона — нумерология как закономерности
- SEO: заголовок, description, slug — обязательны для каждой статьи

## Связанный проект — Бот

Полный контекст бота: `docs/bot-context.md`

Бот и лендинг — два независимых проекта, но единый продукт:
- Лендинг привлекает трафик → конвертирует в пользователей бота
- В перспективе: калькулятор чисел прямо на сайте (без Telegram)
